# GitHub ActionsからFirebaeを使ったAndroid/Webビルドを配布するテスト

このRepositoryでは以下の内容を検証しています。
* GitHub Actions + GameCIを使ったUnityのAndroid/Webビルド
* Firebase App Distributionを使ったAndroidビルドの配布
* Firebase Hostingを使ったWebビルドの配布
